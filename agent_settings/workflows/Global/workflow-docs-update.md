# Documentation Update Workflow

Based on `805-docs-update-standards-specific.md` and `840-docs-readme-specific.md`.

1. **Timestamp Update**
    - Ensure `<!-- Generated: ... -->` timestamp is current if present.

2. **Content Update**
    - Use specific references (file paths, line numbers).
    - Be concise.

3. **Link Verification**
    - Check that relative links point to existing files.

4. **Catalog Update**
    - Update `docs/files.md` if new files were added.
    - Update `README.md` features section if applicable.
